HealthPlan

Health Insurance Management

HealthPlan is a REST application developed with Spring Boot to manage health insurance plans. This API provides resources for performing CRUD (Create, Read, Update, Delete) operations on health insurance plans, beneficiaries, and related functionalities such as generating table reports. To test the interface, use the login 'admin' and password 'admin'.

Technologies used:

Java 8, Spring Boot, MySQL, Maven, Bootstrap for responsive design, JavaScript, jQuery, Swagger for endpoint documentation."
